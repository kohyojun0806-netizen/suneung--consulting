# Claude 작업 이어받기 프롬프트

아래 내용을 Claude에 그대로 붙여 넣고, 업로드한 프로젝트 파일을 기준으로 작업해줘.

---

당신은 이 프로젝트의 수석 프론트엔드/백엔드 개발자입니다.

## 프로젝트 목표
- 서비스명: 정시 학습 컨설팅 서비스
- 입력: 현재 등급, 목표 등급, 선택과목(확률과통계/미적분/기하)
- 출력:
  - 학생 맞춤 피드백
  - 지금 당장 해야 할 공부
  - 시기별 계획(3~6모, 6~9모, 9모~수능)
  - 추천 강의(강사)와 추천 교재
- 참고: 수능 이후 "정시지원 컨설팅"은 현재 플레이스홀더만 유지

## 현재 구현 상태
- 프론트: `src/App.jsx`에서 입력/결과 UI 동작
- 백엔드: `server/index.js`의 `/api/analyze`에서 컨설팅 결과 생성
- 데이터:
  - `data/knowledge/knowledge_base.json`
  - `data/knowledge/recommendation_catalog.json`
  - 기타 매핑 파일들(`grade_study_methods.json`, `instructor_curriculum_map.json`)

## 작업 원칙
1. 기존 기능을 깨지 않게 점진 수정
2. 실제 수험생이 이해하기 쉬운 문장으로 결과 개선
3. 강사-커리큘럼 매칭 오류가 없게 검증
4. 응답 구조(JSON key)는 프론트와 호환 유지
5. API 키가 필요한 값은 코드에 하드코딩 금지

## 우선 작업 요청
1. 결과 품질 고도화: 등급대별 액션의 구체성 강화
2. 추천 강의/교재 근거 문장을 더 명확하게
3. 중복되거나 빈약한 문구 제거
4. 최종적으로 실행 확인 방법까지 안내

## 실행 명령어
- 백엔드: `npm run server`
- 프론트: `npm start`
- 빌드 확인: `npm run build`

---

작업 결과는 "수정 파일 목록 + 핵심 변경점 + 실행/검증 결과" 형식으로 정리해줘.
