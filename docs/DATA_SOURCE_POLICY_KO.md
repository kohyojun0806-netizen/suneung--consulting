# 학습 데이터 수집 기준 (정시 수학 컨설팅)

## 1) 목표
- 등급별/시기별 학습법 추천 품질 향상
- 강사/강의/교재 추천의 정확도 향상
- 1회성 답변이 아닌 누적 코칭 품질 개선

## 2) 우선 수집 출처
- 입시 커뮤니티: 오르비, 포만한, 수만휘
- 영상: 유튜브 수학 등급별 학습법/커리큘럼/OT
- 강사 공식 페이지: 메가스터디, 대성마이맥
- 학원 페이지: 시대인재, 두각학원
- 공신력 문서: 한국교육과정평가원(KICE)

## 3) 수집 대상 데이터
- 학습법 칼럼/게시글/댓글: 실패 원인, 개선 루틴, 복기 방식
- 수강 후기: 강의 난이도, 속도, 대상 등급, 추천 선행
- 공식 커리큘럼/OT: 강좌 순서, 교재명, 시즌 운영 방식
- 시기별 전략: 3~6월, 6~9월, 9월~수능 전

## 4) 분류 버킷
- `study_methods`: 근본 학습법, 문제 접근, 오답 체계
- `lecture_books`: 강사/강의/교재/커리큘럼
- `learning_routines`: 주간 루틴, 누적 관리, 실행 체크리스트

## 5) 품질 체크
- 동일 주장 최소 2개 이상 출처 교차 확인
- 후기성 글은 과장 가능성 표시(단정 금지)
- 특정 강사 추천 시 대상 등급/성향 명시
- 오래된 정보(커리큘럼 개편 가능)는 최근 시즌 정보 우선

## 6) 운영 원칙
- robots.txt / 서비스 약관 준수
- 로그인/유료/비공개 영역은 수집 대상 제외
- 개인정보 포함 댓글/게시물은 저장 금지
